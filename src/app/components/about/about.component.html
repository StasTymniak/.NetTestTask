Main step in my algorithm:

  • Take English alphabet and all digits "ABCDEFGHIJKLMNOPQRSTUVWXYZ1234567890"
  • Using Random get 8 chars and create randStr
  • Using HttpContext.Request.Scheme and HttpContext.Request.Host plus our randStr create shorten version of provided link
  • Map our shorten link and original link in database using Url model
  • When we want to use out shorten link, we will trigger HttpGet request which find our short link in database and redirect to original link
